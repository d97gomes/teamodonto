
# TeamOdonto
## Sistema Web para Gestão de Clínica Odontológica

## 1. Identificação do Projeto
O TeamOdonto é um sistema web desenvolvido como projeto acadêmico para a disciplina de Desenvolvimento Web, com foco no gerenciamento de clínicas odontológicas.

## 2. Objetivo
Aplicar na prática conceitos de desenvolvimento web utilizando arquitetura MVC, integração frontend e backend, controle de sessão e persistência de dados.

## 3. Escopo
- Autenticação de usuários
- CRUD de Pacientes
- CRUD de Dentistas
- Anamnese do paciente
- Estrutura para agenda, consultas e orçamentos

## 4. Arquitetura
O sistema utiliza arquitetura MVC (Model-View-Controller).

Fluxo:
View → Axios → index.php → Controller → Model → MySQL

## 5. Tecnologias
- PHP 8.x
- MySQL
- HTML5 / CSS3
- JavaScript
- Bootstrap 5
- Axios
- Git
- XAMPP

## 6. Segurança
Controle de acesso por sessão PHP, protegendo rotas e APIs.

## 7. Estrutura de Pastas
app/controllers, app/models, app/views, public/js, public/css

## 8. Módulo Paciente
Gerenciamento completo de pacientes com dados pessoais, endereço e histórico clínico.

## 9. Módulo Dentista
Gerenciamento de profissionais com dados pessoais e profissionais.

## 10. Anamnese
Registro clínico associado ao paciente, contendo histórico de saúde e observações.

## 11. API REST
API REST em PHP utilizando métodos GET, POST, PUT e DELETE.

## 12. Versionamento
Versionamento realizado com Git.
